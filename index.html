Welcome to my website. I like to code . Below is a simple demonstration on how altering the standard deviation affects the normal distribution. Notice how as you increase sd, the data tends to deviate more. This is because the definition of sd is the mean average 'distance' a data point is away from the mean. The seed value determines how the data is randomly generted. This demonstration was coded in r and deployed using the shiny package.
