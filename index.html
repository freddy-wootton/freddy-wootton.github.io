I like to code. Below is a simple demonstration of how altering the standard deviation affects the normal distribution. Notice how as you increase the standard deviation, the data tends to deviate more. This is because the definition of standard deviation is the mean average 'distance' a data point is away from the mean. The seed value determines how the data is randomly generated. This demonstration was coded in R and deployed using the Shiny package.
